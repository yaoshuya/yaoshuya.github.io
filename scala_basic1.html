    Scala背景

    scala 代表 scalable language,可扩展的语言,这是它诞生的主要使命. 其作者 martin odersky在2003研发出来,它是一种高性能,高并发,集函数式与OO于一身的编程语言, 运行于JVM之上,可以与java进行无缝整合.

函数

在函数式编程语言中,纯函数需要满足如下的条件:

  • 有一个或多个输入参数
  • 仅使用输入参数进行计算
  • 返回一个值
  • 对于相对的输入值,返回值是固定的
  • 不修改函数外的任何值
  • 不受函数外的值的任何影响

过程

没有返回值的函数称为过程,其最后的代码为语句

def add(a : Int*) : Int = {
    var total = 0
    for ( i <- a ){
       total += i
    }
    total
  }

常用方法

  • :: 1 :: 2 :: Nil Nil连接构建列表
  • ::: List(1, 2) ::: List(2, 3) 从头部追加为一个新的列表
  • ++ List(1, 2) ++ Set(3, 4, 3) 从尾部追加为一个新的列表
  • == List(1, 2) == List(1, 2) 如果两个列表值相同则返回true
  • distinct List(3, 5, 4, 3, 4).distinct 返回一个没有重复内容的列表
  • drop List('a', 'b', 'c', 'd') drop 2 去掉前两个元素
  • filter List(23, 8, 14, 21) filter (_ > 18) 返回一个满足满足过滤条件的新列表
  • flatten List(List(1, 2), List(3, 4)).flatten 把嵌套的列表压平为一级列表
  • partition List(1, 2, 3, 4, 5) partition (_ < 3) 对列表元素进行分组,变成两个列表
  • reverse List(1, 2, 3).reverse 逆向排列列表元素,新的列表
  • map List("milk","tea") map (_.toUpperCase) 把函数应用于列表中的每一个元素,返回一个新的列表,示例为返回一个元素都转为大写的新列表

列表reduce类操作

  • foldLeft List(4, 5, 6).foldLeft(0)( + ) 以初始值0从左到右把列表元素累加
  • foldRight List(4, 5, 6).foldRight(0)( + ) 以初始值为0从右到左把列表累加
  • reduce List(4, 5, 6).reduce( + ) 返回列表元素累加值

类和对象

apply方法

它是类的默认方法,不需要显式调用其名字就可以触发,你使用className(),就触发了apply()方法. 它常用于类的伴生对象构建类的实例.

unapply方法

它是用于模式匹配的,可以把类的成员变量解析出来用于模式匹配.

Object

在scala中,Object是类似java中的静态类,它是单例的. 类的伴生对象 在同一个scala文件里,和class名字相同的object称为伴生对象,可以访问class的私有成员变量和方法.

case class 样例类

  • 样例类是编译器自动生成了一些方法,生成其伴生对象.还有如apply,unapply方法.
  • apply方法使用方便构建,直接使用className(...)构建对象实例.
  • unapply方法使其可以用在模式匹配中.
  • 通过反编译其代码来看,还有其他的一些操作.
  • 样例类主要用于存储数据,类似pojo,用于数据传输和消息传送(Akka框架中)

Traits (特征 特点)

  • trains的作用类似java中的接口,它主要用于多继承,其中的函数可以有默认的实现.
  • 如果一个类继承了两个Trains,而且这两个Traits有相同的字段或方法,若不使用重载的话, 编译是无法通过的.
  • 多继承的时候,是从右到左的,以下代码的示例可以说明
  • selfType 限定Traits只能被特定的类使用
    trait base { override def toString = "Base" }
    class A extends base{ override def toString = "A->" + super.toString}
    trait B extends base { override def toString = "B->" + super.toString }
    trait C extends base { override def toString = "C->" + super.toString }
    class D extends A with B with C { override def toString ="D->" + super.toString()}
    println(new D())

隐式类与隐式参数

隐式类类似于c++里面的转换构造函数,可以把一个类转换成另一个类.

  • 常用于构造工具类,给我们常用的类定义一些有用的方法,然后直接调用,像这个类拥有了这个方法一样.
  • 隐式参数用于部分调用函数的时候,implicit修饰的参数会有调用者提供的implicit变量提供或显式地提供.具体请见示例代码.
    object Doubly {
        def print(num: Double)(implicit fmt: String="%.2f") = {
          println(fmt format num)
        }
      }
      implicit val fmt = "%.1f"
      Doubly.print(2.34333)
      Doubly.print(2.34333)("%.3f")
